Risk Assessment for a CYBER CAFÉ 

CRITICAL ASSETS 
1. Customer data (login details, files, browsing activities)
2. Admin access to all systems 
3. Business financial records
4. Printer and shared storage devices 

POTENTIAL THREATS 
1. Malware infections from USB drives or downloads
2. Customers stealing or viewing other users' data
3. Unauthorised access to admin system 
4. Phishing attacks on admin's email

VULNERABILITIES 
1. Outdated antivirus software
2. Weak admin password 
3. Lack of monitoring of user activities 
4. No user restrictions 

RISK EVALUATION 
1. Malware from USB/downloads is high
2. Data theft between users is high
3. Weak admin password is high
4. Phishing attacks on admin is medium

MITIGATION STRATEGIES 
1. Install an antivirus software on all systems 
2. Use strong admin password and enable 2 or multi factor authentication 
3. Set up a guest user account 
4. Clear browser history and files after every session 
5. Install software to scan USB devices
